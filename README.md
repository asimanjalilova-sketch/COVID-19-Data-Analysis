A Python data analysis project exploring the progression of COVID-19 cases in Germany using pandas, matplotlib, and data from the COVID-19 Dataset by Our World in Data

This project performs a full analysis workflow:

Loads live COVID-19 data from GitHub

Cleans and preprocesses it

Calculates daily confirmed cases, deaths, and active cases

Visualizes key metrics such as daily trends and recovery patterns

Exports clean data and summary statistics to CSV
